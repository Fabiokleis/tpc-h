# tpc-h

Projeto 1 - TPC-H em MongoDB

Executar as 3 primeiras queries do TPC-H em um banco de dados MySQL.

Carregar os dados para collections do MongoDB, com liberdade para alterar a estrutura dos dados (incluindo repetição) da maneira que se achar melhor.

Gerar um relatório com a comparação dos tempos de execução.
